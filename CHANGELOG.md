# Changelog

All notable changes to `laravel-barangay-search` will be documented in this file.

## v1.0.0 - 2024-XX-XX

### Added
- Initial release
- Livewire 3 component for Barangay Search
- Integration with GIS.PH SDK
- Mary UI support
- Vanilla styling option
- Intelligent caching
- Filtering by municipality, city, province
- Events for selection, clearing, and errors
- Comprehensive configuration options
- Service class for direct API usage
- Full test coverage
